# 深夜酒類 図面エディタ

深夜酒類提供飲食店営業開始届出（深夜0時以降にお酒を出す飲食店の届出）で必要になる
**平面図・求積図・照明音響図・届出書類の下書き** を、ブラウザだけで作れる専用エディタです。

汎用CADではなく、深夜酒類の届出に必要なことだけを正確にやる道具を目指しています。

## 特徴

- **インストール不要** — `index.html` をブラウザで開くだけで動きます（依存ライブラリゼロ）。
- **数値入力で正確に作図** — 店舗の外形・区画を mm 単位で入力。10cm グリッドに吸着。
- **多角形の区画** — L字・斜め壁など自由な形を、角をクリックして囲うだけで作図。面積は**座標法**で自動計算し、座標求積表（X・Y座標の計算表）を図面に添付。頂点はドラッグや数値入力で修正でき、角度も入力で調整可。
- **建具を作図** — 出入口・扉・戸・窓・壁を製図記号で配置（平面図に表示、角度調整可）。扉・戸は**片開き扉・両開き扉・片引き戸・引き分け戸・引き違い戸**から選べ、**開き勝手（吊元の左右・内/外開き）**をチェックで切り替えられる（引戸はスライド方向を矢印で表示）。柱は区画として追加する。
- **自由な形の備品** — テーブル・カウンター等を、区画の多角形と同じ要領で**真上から見た形をクリックでなぞって**作図。カウンター姿図は薄い天板・本体・蹴込みを分けて表示。L字カウンターや変形テーブルなど、定型にない形も配置できる。頂点はドラッグ・座標入力で修正可。さらに**備品姿図の画面で「正面の形を描く／側面の形を描く」**を押すと、横から見たシルエットも点でなぞれる（描いた形は姿図にそのまま表示。輪郭の最高点に高さが自動で合い、1m超なら見通し規制の注意色になる）。
- **壁芯求積に対応** — 営業所外周（壁厚つき）を描くと壁芯線を自動生成。営業所求積は「区画の合計（内法）/ 壁芯の外周（座標法）/ 両方併記」から警察署のローカルルールに合わせて選択可。
- **慣行色の色分け** — 求積図の線色を「標準（黒）/ 色分け（営業所=青・客室=赤・調理場=緑）」で切り替え可。色分け時は営業所・客室・調理場の囲い線を、実線・点線・破線・非表示から選べる。
- **求積表を自動生成** — `3.20 × 4.50 = 14.4000 ㎡` のような計算式と合計を自動で出力。
- **面積囲い線** — 客室・調理場・営業所の範囲を、色と線種を選んで多角形で囲える。囲った範囲は塗りなしの線だけで表示され、その面積が求積表・面積サマリーに反映される。面積囲い線がある場合は、通常区画ではなく囲い線を求積表の対象にし、計算式の表示文は手入力で調整できる。辺の長さ表示と頂点番号(P1...)は面積囲い線ごとにオン/オフできる。
- **面積の扱いを選択** — 区画の見た目とは別に「客室に算入」「調理場に算入」「営業所にのみ算入」「表示のみ」を選択可。カウンター区画は初期状態で客室面積に入る。カウンター備品の位置も営業所求積図・客室/調理場求積図に表示される。
- **図面上の表を調整** — 営業所求積図・客室/調理場求積図・照明音響設備図の右下表は、初期80%サイズで表示。クリックしてドラッグ移動・右下ハンドルでサイズ調整できる。
- **区画寸法の自動表示** — 営業所求積図・客室/調理場求積図では区画の長さを自動表示。客室/調理場求積図でも通路・トイレ・柱などの寸法を確認できる。
- **面積サマリー** — 営業所・客室・厨房・トイレ・その他の面積を自動集計。
- **備品の番号と姿図** — 備品にサイズ違いを区別する番号（同じ種類・同じ寸法は同じ番号、寸法が違うと②③…）を自動付与。正面図・側面図を寸法つきで並べた「備品姿図」を自動生成し、床から1mの基準線（見通し規制）と備品一覧表を添付。各品は同じ大きさのカードに整え、**用紙の幅に合わせて自動で段組み（折り返し）**するので、種類が増えても用紙からはみ出しにくい。
- **図面の切り替え** — 平面図 / 営業所求積図 / 客室・調理場求積図 / 照明・音響設備図 / 備品姿図。
- **照明・音響設備一覧表** — 記号・数量・ワット数を自動集計して設備図に添付。数量はサイドバーから手入力で補正可。
- **要件チェック** — 見通し規制（高さ1m超）と客室床面積（2室以上は各9.5㎡以上）を自動で注意表示。施錠・照度20ルクス等のリマインダーつき。
- **届出書類の下書き** — 営業開始届出書（様式第47号）・営業の方法（様式第48号）を、図面から面積・客室数・設備を自動転記して生成。下線部はその場で編集可。
- **必要書類チェックリスト** — 住民票・定款（法人）・許可証写し等の添付書類を個人/法人別に管理。
- **縮尺表示つきPDF** — 1枚ずつ、または届出に必要な**図面一式を一括**でPDF出力。平面図・営業所求積図・客室/調理場求積図には、用紙枠内と同じ形式のスケールバーと縮尺を表示。
- **保存・読み込み** — 図面とチェックリストの状態を JSON ファイルで保存・再開。
- **案件ごとの自動保存** — 変更のたびにブラウザ内へ自動保存し、リロード・再訪問時に続きから自動復元。複数の案件（店舗）を左パネルの「案件」欄で切り替えて並行作業できる。ブラウザ・端末ごとの控えのため、大事な案件はファイル保存も併用を推奨。
- **オフライン対応** — 一度オンラインで開くと、次回以降はネットがない場所でも起動できる。Chrome/Edgeではアプリとしてインストールして使える。
- **元に戻す / やり直す** — Cmd/Ctrl+Z（Shift付きでやり直し）またはツールバーのボタンで、直前の操作を取り消し。
- **要素の複製** — Cmd/Ctrl+D または「複製」ボタンで、選択中の区画・備品・建具・設備・メモを一発コピー（同寸法の備品は同じ番号になる）。
- **選択補助** — 「選択対象」で寸法線だけ・面積囲い線だけなどに絞り込める。右パネルの要素一覧からクリック選択・削除もでき、重なった要素を確実に編集できる。
- **重なり順の調整** — 選択中の要素を「最前面・前面・背面・最背面」へ移動できる。要素一覧で赤線や文字を選んでから前面に出せるので、線や文字が重なって見えないときに調整しやすい。
- **メモ・コメント** — 図面の好きな場所に注記を配置。メモは矢印と囲いあり、コメントは囲いなしの自由テキスト。複数行・文字サイズ調整可。図面ごとに表示され、PDFにもそのまま印字。
- **手動の寸法線** — 任意の2点をクリックして長さ(m)を記入。平面図・営業所求積図・客室調理場求積図に共通表示され、壁から客室までの距離・出入口幅など審査で求められる寸法に対応。両端ドラッグ・複製可。
- **実寸の縮尺でPDF印刷** — PDFを用紙に「収める」のではなく、定規で測って正確に1/縮尺になる物理サイズで出力するオプション(印刷倍率100%）。用紙からはみ出す場合は警告。
- **届出書への自動転記(拡充)** — 「届出者情報」に届出者の住所・氏名・電話・建物構造・営業時間等を入力すると様式第47号・第48号へ自動転記。営業所床面積は求積方式(壁芯/内法)に追従、宛先の公安委員会は所在地から自動推定。
- **全案件の一括バックアップ** — ブラウザ内の全案件を1つのJSONファイルへ書き出し / 取り込み（パソコン買い替え・ブラウザ変更時のデータ移行用）。
- **下絵トレース** — 間取り図や手描きスケッチの画像を背景に薄く表示し、その上をなぞって作図。横幅(mm)指定で縮尺合わせ、濃さ調整・ドラッグ移動可。下絵はPDFには出力されない。
- **柱の面積控除** — 区画として追加した柱が立っている区画（客室・調理場など）の求積表に「△0.09㎡」のような控除行を入れ、合計から差し引くオプション（警察署の扱いに合わせてオン/オフ）。

## 使い方

1. `index.html` をブラウザ（Chrome / Edge / Safari など）で開く。
2. 左パネルから区画（客室・厨房・トイレ等）を幅・奥行を入れて追加。
3. キャンバス上でドラッグして配置（既定 1cm 単位で吸着、Shift で自由移動）。
4. 備品・照明音響設備を追加して配置。
5. 上部のタブで図面の種類を切り替える。
6. 右パネルで求積表・面積・注意点を確認。
7. 図面情報で縮尺（1/50・1/80・1/100）と用紙を選ぶ。
8. 「図面一式PDF」で提出用の全図面を一括出力（「この図面をPDF」で1枚ずつも可）。
9. 「届出書作成」で様式第47号・第48号の下書きを生成し、空欄を埋めて印刷。
10. 「必要書類」で添付書類のチェックリストを確認（個人/法人切り替え）。

### オフライン利用

- GitHub Pages版は、一度オンラインで開くと必要ファイルをブラウザに保存します。次回以降はネットがない場所でも同じURLから起動できます。
- Chrome/Edgeではアドレスバー付近のインストール操作から、アプリのように起動できます。
- 完全にネットなしで初回利用したい場合は、このフォルダ一式をZIPやUSBで持ち出し、`index.html` をブラウザで開いてください。
- 案件データは端末内のブラウザに保存されるため、現場へ行く前に「全案件をバックアップ」でJSONを書き出しておくと安全です。

## 届出に必要なもの（本ツールの対応範囲）

| 書類 | 対応 |
|---|---|
| 営業開始届出書（様式第47号） | ✅ 下書き生成（面積・客室数・設備を自動転記） |
| 営業の方法（様式第48号） | ✅ 下書き生成 |
| 営業所平面図 | ✅ 作図・PDF出力 |
| 営業所求積図・求積表 | ✅ 作図・PDF出力 |
| 客室・調理場求積図・求積表 | ✅ 作図・PDF出力 |
| 照明・音響設備図（一覧表つき） | ✅ 作図・PDF出力 |
| 営業所周辺の略図（案内図） | ❌ 対象外（住宅地図等を利用）。チェックリストで管理 |
| 住民票・定款・登記事項証明書・許可証写し等 | チェックリストで管理 |

提出先は営業所所在地を管轄する警察署（公安委員会宛て）、期限は営業開始の **10日前まで**。
必要書類・様式は都道府県警により異なるため、必ず最新の案内を確認してください。

### 操作
- 空きをドラッグ … 画面の移動（パン）
- ホイール … 拡大・縮小
- 要素をドラッグ … 配置（10cm 吸着 / Shift で自由）
- Delete / Backspace … 選択要素を削除

## 構成

```
index.html          画面
manifest.webmanifest PWAのアプリ情報
service-worker.js   オフライン起動用キャッシュ
css/style.css       スタイル
js/model.js         データモデル・カタログ・チェックリスト・保存読込
js/geometry.js      求積（面積計算）・要件チェック・設備集計
js/render.js        Canvas 描画・レイヤー
js/interactions.js  マウス操作
js/print.js         PDF / 印刷（1枚・図面一式）
js/forms.js         届出書類（様式第47号・第48号）の下書き生成
js/main.js          画面の組み立てと配線
```

## 現状の制限（今後の予定）

- PDF はページに収めて出力します（プリンタ・ブラウザ差により正確な縮尺印刷は保証していません。縮尺は表題欄に表記）。
- 多角形の求積は座標法です。「縦×横」の計算式が必要な場合は、長方形・三角形の組み合わせで表現してください（自動三角形分割は将来検討）。
- DXF / Jw_cad 連携、3D 表示は未対応（将来検討）。

## 注意

本ツールは作図を補助するものです。実際の届出にあたっては、所轄警察署・公安委員会の
最新の様式・要件を必ずご確認ください。
